About TheMealDB

Description

This web application allows users to discover recipes from around the world using different search criteria. Users can search by name, category, ingredient, geographic area, or simply discover random recipes.

The main objective is to provide a simple and accessible way to explore new culinary ideas and expand gastronomic knowledge.

Used API

This application consumes the TheMealDB API, an open database containing thousands of recipes from around the world.

TheMealDB provides detailed information about dishes, including:

  • Names and images
  • Lists of ingredients and measures
  • Step-by-step instructions
  • Categories and geographic areas
  • Links to instructional videos when available
